## 运行

编译完成后得到可执行文件，执行即运行了C代码。

对于单源码文件编译成的程序来说，程序的入口是`main`函数。`main`函数需要以如下方式定义：

```c
int main(void) {
    // 一些代码
    return 0;
}
```

在后面的教程中会涉及到函数的有关概念。`main`函数是允许没有形参、返回整形类型的函数，其返回值将会作为程序进程结束时的返回值，`0`表示程序正常结束，任何非`0`的值都意味着错误。
