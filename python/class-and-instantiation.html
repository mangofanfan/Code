什么是面相对象？

在阅读下面的教程前，你需要首先了解什么是，以及为什么我们需要面向对象。
如果没有的话，可以看看这里o.O

Python使用class关键字来定义类型。类型是Python中很常见的东西，这也符合Python的设计理念。

python
class Cat:
    def __init__(self, name: str, age: int)
        self.name = name
        self.age = age
    def miaow()
        print(f"{self.name}: Miaow Miaow!!")

cat = Cat("XiaoFan", 1)
cat.miaow()
