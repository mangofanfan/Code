# HTML

## 概述
**HTML**（超文本标记语言，HyperText Markup Language）是用于创建网页的标准标记语言，用于描述网页结构和其上存在的元素。

HTML 是一种标记语言，而非一种编程语言。在 Web 前端使用的编程语言是 JavaScript，请查看 [JavaScript 特性](/web-front/javascript/features)。

HTML 只负责描述元素的存在，不负责描述元素的样式。在 Web 前端负责定义元素样式的语言是 CSS，请查看 [CSS 特性](/web-front/css/features)。

HTML 文件的后缀名为 `.html`。有些时候也被简写为 `.htm`。

## 浏览器开发者工具
对于桌面平台的大部分浏览器，你可以使用键盘上的 `F12` 按键打开它们的开发者工具。

浏览器的开发者工具包含大量的功能，基础的包括显示网站的 HTML 源码、查看 CSS 样式、查看 JavaScript 脚本，以及一个可交互的 JavaScript 控制台、网络请求管理、网络资源管理等等。

由于现阶段的网页技术是在浏览器中使用 HTML CSS JavaScript 三件套实现用户端渲染的，因此在网站上存在的任何东西都是可以被用户查看的。因此，Web 开发中所谓的禁止用户复制代码、禁止用户查看源码等，都是没有意义的行为。
