# Web 开发

:::warning Web 开发 => Web 前端？
若无特别强调，**Code Space 中的「Web 开发」部分主要介绍的内容为 Web 前端。**

实际上完整的 Web 包括前端和后端。在 Code Space 中，我们将 Web 后端划归回到它们各自的编程语言中，例如 Java 的 Spring Boot、Python 的 Django 等。
:::

## Web 基本原理
学习掌握 Web 开发是一项入门简单、学习难度陡峭、好玩程度拉满的活动。见证一个名为浏览器的应用程序在一个空白的窗口中渲染出你亲手写就的页面是一件非常有成就感的事，在那时一切你先前披星戴月的付出、遇到问题的迷茫、生啃文档的晦涩，都会化为发自内心的喜悦。

掌握 Web 开发需要我们了解很多，但幸运的是，我们使用 Web 之力并不需要首先完全掌握 Web 开发。

在 Code Space 的「Web 开发」部分的**左侧边栏**中，你可以找到经过编排的教程列表，其已经进行了基本的归类。
